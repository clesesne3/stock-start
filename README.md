# Stock-Start

### Problem Domain

Addresses individuals who are hesitant or desire more information about investing in the stock market.

### User Stories

1. Simple, welcoming experience that encourages those to pursue stock investments
2. Personality, financial goals, and risk tolerance match appropriate stocks
3. Provided with quick access to current stock quotes
4. Provided with links to suggested online investing/trading platforms

### Technical Features

- Links to current stock quotes that are matched to user
- Links to popular investment/trading platforms
- Database table with stocks of varying risk
- Ability to filter and display stocks based on user profile
